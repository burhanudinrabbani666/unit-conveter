# Unit Converter

A beginner project built with Node.js and Express — convert between units of length, weight, and temperature.

> Project from: https://roadmap.sh/projects/unit-converter

## Features

- **Length** — millimeter, centimeter, meter, kilometer, inch, foot, yard, mile
- **Weight** — milligram, gram, kilogram, ounce, pound
- **Temperature** — Celsius, Fahrenheit, Kelvin

## Getting Started 🚀

**Prerequisites:** Node.js installed on your machine.

```bash
git clone https://github.com/burhanudinrabbani666/unit-conveter

cd unit-conveter

npm install

npm start
```

Then open your browser at `http://localhost:3000`.
